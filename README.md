# Shopify Umino Theme

## Project Description
The Shopify Umino Theme is an elegant and responsive theme designed to provide an exceptional shopping experience. It is built with scalability in mind, ensuring compatibility with various Shopify apps and customizations.

## Objectives
- Provide a seamless and intuitive user experience.
- Ensure mobile responsiveness and quick loading times.
- Incorporate best practices for SEO.
- Allow for easy customization through the Shopify admin panel.
- Support both LTR and RTL languages.

## File Structure
```plaintext
/  
├── assets/           # Theme assets (images, styles, scripts)
├── config/           # Theme configuration files
├── locales/          # Language translations
├── sections/         # Reusable sections of the theme
├── snippets/         # Small reusable code blocks
├── templates/        # Template files for various pages
└── config.yml       # Main configuration file for the theme
```

## Getting Started Guide
1. Download the theme files.
2. Go to your Shopify admin panel.
3. Navigate to `Online Store` -> `Themes`.
4. Upload the theme files.
5. Customize your theme settings as per your requirements.

## RTL Support Instructions
To enable RTL support:
1. Open `config.yml`.
2. Add the following configuration:
   ```yaml
   rtl: true
   ```
3. Adjust any stylesheets accordingly to ensure proper alignment and styling.
